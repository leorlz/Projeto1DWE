Mapa das Escolas Municipais de Campinas

Este projeto foi criado para a disciplina de desenvolvimento web sob pedido do Prof. André Bordignon. Seu objetivo é listar algumas das escolas municipais de campinas em um mapa geo-referenciado.
Se você sabe de alguma escola que não está no mapa ou encontrou alguma informação incorreta, por favor, ajude-nos a melhorar este projeto utilizando nossa página de contato
